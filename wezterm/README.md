# WezTerm Configuration

## Dependencies
```bash
wezterm
```

## Configuration
- **Font**: JetBrains Mono
- **Opacity**: 0.88
- **Tab bar**: Hidden when single tab
- **Scrollbar**: Enabled

## Files
```
~/.config/wezterm/
└── wezterm.lua
```

## Default Keybindings
- `Ctrl+Shift+Space` - Quick select
- `Ctrl+Shift+F` - Search
- `Ctrl+Shift+C/V` - Copy/Paste
- `Ctrl+Shift+N` - New window
- `Ctrl+Shift+T` - New tab
- `Ctrl+Tab` - Next tab
